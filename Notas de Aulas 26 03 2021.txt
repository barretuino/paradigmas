- Conceitos Importantes da Orientação a Objetos
	* Herança
		- Extensão: quando uma classe extende outra classe. Exemplo
			public class Aluno extends Pessoa{
				....
			}
			herança do tipo "É um"
		- Composição: quando a classe contém implementações de atributos
		que são objetos de um outro tipo de classe.
		
	* Métodos: Conceito Polimorfismo que é implementado por meio de 
	um técnica denominada Sobrecarga de Métodos.
		Assinatura do método: parâmetros ou argumentos que o método
		(estruturada é função)
		1. Quantidade de Argumentos
		2. Tipo dos Argumentos
		3. Ordem dos Argumentos

API:
    Software para Vendedores: Módulo de cadastro de Pedidos de Venda
